# anthro690_finalproject
Final project for Anthro 690

Note on files:
- all_activity.csv is the original dataset used for the project
- monkey_subject.csv is the cleaned dataset used for analysis
- code.R is the code used for cleaning and inferential stats, contains figs 3 and 6
- descriptive_stats.R is the code used for descriptive stats, contains figs 1 and 2
- figures.R contains code for earlier visualizations
- violin_plots.R contains code for making various violin plots
- final_plots.R contains code for figures 4, 5 and 7 from the paper
- Figures folder contains all versions of figures, including those used in the paper and earlier versions
